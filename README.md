# Installers

Mac style installations with symbolic link.

## Usage

```bash
cd /tmp
rm -f download.sh*
wget https://raw.githubusercontent.com/wbotelhos/installers/master/download.sh
chmod +x download.sh
sudo ./download.sh
cd installers
```

And then, check the README.md inside each installer.
